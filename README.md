# stub-maven-plugin

## usage

```xml
<plugin>
  <groupId>org.f4</groupId>
  <artifactId>stub-maven-plugin</artifactId>
  <version>1.0-SNAPSHOT</version>
  <executions>
    <execution>
      <phase>process-classes</phase>
      <goals>
        <goal>stub</goal>
      </goals>
    </execution>
  </executions>
</plugin>
```
